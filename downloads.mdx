## Installation Instructions

<Tabs>
  <Tab title="macOS">
    <Steps>
      <Step title="Download">
        Visit the [macOS releases page](https://github.com/imputnet/helium-macos/releases/latest) and download the latest `.dmg` file.
      </Step>
      <Step title="Install">
        1. Open the downloaded `.dmg` file
        2. Drag Helium to your Applications folder
        3. Launch Helium from Applications or Spotlight
      </Step>
      <Step title="First Launch">
        On first launch, macOS may show a security warning. To open Helium:
        1. Go to System Settings > Privacy & Security
        2. Scroll down to find the message about Helium
        3. Click "Open Anyway"
        
        Alternatively, right-click Helium in Finder and select "Open".
      </Step>
    </Steps>
    
    <Note>
      Helium for macOS is a universal binary that runs natively on both Intel and Apple Silicon Macs.
    </Note>
  </Tab>
  
  <Tab title="Linux">
    <Steps>
      <Step title="Download">
        Visit the [Linux releases page](https://github.com/imputnet/helium-linux/releases/latest) and download the latest `.AppImage` file.
      </Step>
      <Step title="Make Executable">
        Open a terminal in your downloads folder and run:
        ```bash
        chmod +x Helium-*.AppImage
        ```
      </Step>
      <Step title="Run">
        Double-click the AppImage file or run it from the terminal:
        ```bash
        ./Helium-*.AppImage
        ```
      </Step>
      <Step title="Optional: Install AppImageLauncher">
        For better desktop integration, consider installing [AppImageLauncher](https://github.com/TheAssassin/AppImageLauncher). It will automatically integrate AppImages into your application menu.
      </Step>
    </Steps>
    
    <Note>
      AppImage format provides excellent compatibility across different Linux distributions without requiring installation or root privileges.
    </Note>
  </Tab>
  
  <Tab title="Windows">
    <Steps>
      <Step title="Download">
        Visit the [Windows releases page](https://github.com/imputnet/helium-windows/releases/latest) and download the latest installer.
      </Step>
      <Step title="Install">
        1. Run the downloaded installer
        2. Follow the installation wizard
        3. Launch Helium from the Start Menu or desktop shortcut
      </Step>
      <Step title="Windows Security">
        Windows may show a SmartScreen warning. Click "More info" and then "Run anyway" to proceed with installation.
      </Step>
    </Steps>
    
    <Warning>
      Auto-updates are not yet available for Windows builds. Check the releases page periodically for new versions.
    </Warning>
  </Tab>
</Tabs>
