Helium supports all standard Chromium keyboard shortcuts plus several custom shortcuts designed to improve productivity and privacy.

## Helium-Specific Shortcuts

These shortcuts are unique to Helium or have been modified from Chromium defaults:

<Tabs>
  <Tab title="macOS">
    | Shortcut | Action | Notes |
    |----------|--------|-------|
    | `Ōīś + Shift + C` | Copy page URL or Inspect Element | Configurable in Settings ŌåÆ Appearance ŌåÆ Behavior |
    | `Ōīś + Shift + E` | Inspect element | New shortcut for DevTools inspector |
    | `Ōīś + Shift + I` | Open DevTools | Standard DevTools shortcut |
    | `Ōīś + Option + N` | New split tab | Create split-screen view |
    | `Ōīś + Option + I` | DevTools (Alt binding) | Alternative DevTools shortcut |
    | `Ōīś + Option + J` | DevTools Console | Open directly to console |
  </Tab>
  
  <Tab title="Windows/Linux">
    | Shortcut | Action | Notes |
    |----------|--------|-------|
    | `Ctrl + Shift + C` | Copy page URL or Inspect Element | Configurable in Settings ŌåÆ Appearance ŌåÆ Behavior |
    | `Ctrl + Shift + E` | Inspect element | New shortcut for DevTools inspector |
    | `Ctrl + Shift + I` | Open DevTools | Standard DevTools shortcut |
    | `F12` | Toggle DevTools | Standard DevTools toggle |
    | `Ctrl + Shift + J` | DevTools Console | Open directly to console |

<Note>
**Configurable Shortcut**: The `Ctrl/Cmd + Shift + C` shortcut behavior can be changed in Settings ŌåÆ Appearance ŌåÆ Behavior. When "Enable quick page link copying" is enabled, it copies the URL. When disabled, it opens DevTools inspector.
</Note>

## Special Browser Features

### Middle-Click Actions

Middle-click (scroll wheel click) provides quick actions:

- **Middle-click on link**: Open link in background tab
- **Middle-click on tab**: Close tab
- **Middle-click on back/forward**: Open in new tab
- **Middle-click and drag**: Autoscroll (Windows by default, configurable via [flags](/usage/flags))

### Drag and Drop

- **Drag tab out**: Create new window
- **Drag tab between windows**: Move tab to another window
- **Drag tab to side**: Create split view (if enabled)
- **Drag URL to bookmarks bar**: Create bookmark
- **Drag link to tab bar**: Open in new tab

## Customizing Shortcuts

Helium provides limited shortcut customization:

<Steps>
  <Step title="Toggle Copy URL Shortcut">
    Navigate to **Settings ŌåÆ Appearance ŌåÆ Behavior** and enable or disable "Enable quick page link copying with Ctrl+Shift+C"
    
    - **Enabled**: `Ctrl/Cmd + Shift + C` copies the page URL
    - **Disabled**: `Ctrl/Cmd + Shift + C` opens DevTools inspector
  </Step>
  
  <Step title="Configure Middle-Click Autoscroll">
    Use the Middle Click Autoscroll flag in [Chrome Flags](/usage/flags) to enable or disable autoscroll on middle-click
  </Step>
  
  <Step title="Extension Shortcuts">
    Go to **Settings ŌåÆ Extensions** and click "Keyboard shortcuts" to customize extension shortcuts
  </Step>
</Steps>

<Note>
Unlike some browsers, Helium does not support full keyboard shortcut customization. Shortcuts are designed to match Chromium standards while adding productivity enhancements.
</Note>

## Tips and Tricks

<CardGroup cols={2}>
  <Card title="Quick URL Copy" icon="link">
    Use `Ctrl/Cmd + Shift + C` to instantly copy the current page URL to your clipboard. Perfect for sharing links quickly.
  </Card>
  
  <Card title="Tab Search" icon="search">
    Press `Ctrl/Cmd + Shift + A` to search through all open tabs across all windows. Great for finding that tab you opened earlier.
  </Card>
  
  <Card title="Emergency Tab Close" icon="xmark">
    Use `Ctrl/Cmd + W` repeatedly to close tabs quickly. Combine with `Ctrl/Cmd + Shift + T` to reopen if you close one by mistake.
  </Card>
  
  <Card title="Split View" icon="window-restore">
    Drag a tab to the side of the browser window while holding `Option/Alt` to create a split-screen view (if enabled in settings).
  </Card>
</CardGroup>

## Keyboard Navigation

For fully keyboard-driven browsing:

1. Use `Ctrl/Cmd + L` to focus the address bar
2. Type your search or URL
3. Press `Tab` to navigate suggestions
4. Press `Enter` to navigate
5. Use `F6` (or `Ctrl/Cmd + Option + Up/Down`) to move focus to page content
6. Use `Tab` and `Shift + Tab` to navigate links and form fields
7. Press `Space` or `Enter` to activate links
